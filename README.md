# 云盘分享管理工具

一个轻量级的网盘分享链接管理工具，支持百度网盘和夸克网盘的分享链接集中管理、搜索、标签分类、有效期监控和批量操作。

## ✨ 核心功能

### 📁 多源数据导入
- **CSV导入**：支持百度网盘、夸克网盘CSV格式，自动识别和解析
- **格式兼容**：智能识别不同表头格式，无需手动调整
- **批量处理**：一次导入数百条分享记录，自动去重

### 🔗 链接智能管理
- **卡片/列表双视图**：支持卡片展示和表格列表两种查看方式
- **批量操作**：全选、批量删除、批量添加标签
- **高级筛选**：按来源（百度/夸克）、有效期状态、标签、关键词搜索
- **快速编辑**：双击直接编辑分享信息，实时保存

### 🏷️ 标签分类系统
- **自定义标签**：创建任意标签，支持自定义颜色标记
- **右键管理**：右键菜单快速添加/移除标签
- **标签筛选**：点击标签快速筛选相关分享
- **批量标签**：为多个分享批量添加相同标签

### ⏰ 有效期智能监控
- **状态标记**：自动识别有效、即将过期、已失效三种状态
- **颜色提示**：绿色（有效）、橙色（即将过期）、灰色（已失效）
- **有效期计算**：精确显示剩余天数（N天有效）
- **过期提醒**：即将过期的分享会高亮显示

### 🔄 网盘实时同步
- **账号管理**：支持多个百度/夸克网盘账号，独立Cookie管理
- **一键同步**：从网盘直接拉取最新分享列表，自动更新本地数据库
- **状态验证**：实时验证Cookie有效性，显示账号状态
- **增量同步**：只同步新增或更新的分享，避免重复数据

### ⚡ 取消分享功能
- **批量取消**：选择多个分享记录，一键取消网盘端的分享
- **智能处理**：
  - ✅ 成功取消：硬删除本地记录
  - ⚠️ 无分享ID（CSV导入）：添加"取消分享失败"标签
  - ⚠️ Cookie失效：添加"取消分享失败"标签
  - ❌ 网盘端失败：保留本地记录，添加"取消分享失败"标签

### 🎨 用户体验优化
- **暗黑模式**：一键切换亮色/暗色主题，保护视力
- **设置持久化**：所有偏好设置自动保存，刷新不丢失
- **响应式布局**：适配不同屏幕尺寸，操作流畅
- **快捷键支持**：常用操作支持键盘快捷键

## 🚀 快速开始

### 环境要求
- **操作系统**：Windows 7/10/11
- **Python版本**：Python 3.8+
- **磁盘空间**：约50MB可用空间

### 一键启动（推荐）
双击运行 `start.bat`，脚本将自动完成：
1. ✅ 检查Python环境并安装依赖（flask, flask-cors）
2. ✅ 清理旧进程（自动删除app.pid文件）
3. ✅ 启动后端服务（端口5000，静默模式）
4. ✅ 打开浏览器访问主界面

**访问地址**：http://127.0.0.1:5000

> **提示**：首次启动可能需要5-10秒，请耐心等待。启动后可在浏览器中收藏此地址以便下次快速访问。

### 手动启动（调试）
```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python backend/app.py
```

服务启动后，手动访问 http://127.0.0.1:5000

## 📊 数据导入格式

### 百度网盘 CSV 格式
文件需包含以下列（表头不限语言）：
| 文件名 | 链接 | 提取码 | 分享时间 | 有效期 |
|--------|------|--------|----------|--------|

### 夸克网盘 CSV 格式
文件需包含以下列（`分享地址`字段可能为多行文本，程序会自动提取链接）：
| 分享文件上一级目录名称 | 分享名 | 分享地址 | 提取码 | 分享时间 | 浏览次数 | 分享状态 |
|------------------------|--------|----------|--------|----------|----------|----------|

> **注意**：夸克网盘的"分享地址"字段可能包含"链接: https://..."这样的多行文本，程序会自动使用正则表达式提取出实际的URL链接。

## 🔐 网盘同步指南

### 夸克网盘同步
1. **获取Cookie**：登录夸克网盘网页版，按F12打开开发者工具，在Network标签页中复制任意请求的Cookie
2. **添加账号**：在"账号管理"中添加夸克网盘账号，粘贴Cookie
3. **验证状态**：点击"验证Cookie"确认账号可用
4. **同步数据**：在"网盘同步"中选择账号，点击"立即同步"

### 百度网盘同步
1. **获取Cookie**：登录百度网盘网页版（https://pan.baidu.com），按F12打开开发者工具，在Network标签页中复制任意请求的Cookie
2. **添加账号**：在"账号管理"中添加百度网盘账号，粘贴Cookie
3. **验证状态**：点击"验证Cookie"确认账号可用
4. **同步数据**：在"网盘同步"中选择账号，点击"立即同步"

> **重要提示**：
> - Cookie包含敏感信息，请妥善保管
> - 网盘Cookie通常有有效期，过期后需要重新获取
> - 百度网盘Cookie需包含`BDUSS`、`STOKEN`等关键字段
> - 支持同时管理多个账号，可混合百度/夸克平台

## ❓ 常见问题

### 启动问题
**Q: 启动时显示"无法访问该网页"？**
A: 服务可能尚未完全启动。请等待5-10秒后刷新页面，或检查`data/logs/app.log`查看错误信息。

**Q: 启动页面一直显示"检测服务状态"？**
A: 请按以下步骤排查：
1. 检查Python是否安装：`python --version`
2. 检查端口占用：`netstat -ano | findstr ":5000"`
3. 查看应用日志：`data/logs/app.log`
4. 尝试重新双击`start.bat`

**Q: 如何跳过启动页面直接访问？**
A: 等服务启动后（约5-10秒），直接访问：
- 主界面：http://127.0.0.1:5000/index.html

### 使用问题
**Q: 关闭网页后后台进程还在运行？**
A: 正常情况下会自动关闭。如果异常，请：
1. 删除`app.pid`文件
2. 在任务管理器中结束`pythonw.exe`进程
3. 重新启动应用

**Q: 同步时提示Cookie失效？**
A: 网盘Cookie有有效期，请重新获取最新Cookie：
1. 重新登录网盘网页版
2. 按F12打开开发者工具，复制最新Cookie
3. 在账号管理中更新Cookie

**Q: 导入的CSV文件格式不正确？**
A: 请确保文件包含必需的列，可以尝试：
1. 使用模板：从网盘导出的原始CSV文件
2. 检查编码：确保文件使用UTF-8编码
3. 检查分隔符：使用逗号分隔的CSV格式

### 数据问题
**Q: 删除的记录还能恢复吗？**
A: 支持软删除，可通过重新同步恢复。如需彻底删除，请在取消分享功能中操作。

**Q: 标签颜色如何自定义？**
A: 在标签管理界面，点击标签右侧的颜色块即可选择新颜色。
